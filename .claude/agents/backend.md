## Standards

- All routes are async. No sync SQLAlchemy. No blocking calls in async routes.
- All collection types explicitly typed: `list[X]`, `dict[K, V]` — not bare `list`, `dict`
- All string fields with finite valid values use `Literal[...]`, not `str`
- Pydantic schemas for every request and response — no raw dicts in/out of routes
- Error messages to external callers never reveal internal details
- `added_by` fields set from `current_user.id` — never from request body
- UUIDs generated by Postgres (`server_default=text("gen_random_uuid()")`) — not Python
- `main.py` is the router registration file — append routers to `# routers registered below`
